# Toggle Enemies

### Requirements
- [BepInEx](https://thunderstore.io/c/lethal-company/p/BepInEx/BepInExPack/) Version 5.4.2305+

## Installing

1. Install [BepInEx](https://thunderstore.io/c/lethal-company/p/BepInEx/BepInExPack/)﻿
2. Download and extract this mod.
3. Copy ﻿the .dll mod file to your "<Lethal_Company-Steam-Files>/BepInEx/plugins" directory.
4. Run the game.  Enjoy!

## Usage
Modify the entries automatically generated in your BepInEx configuration file path, usually located at [ ./BepInEx/config/SpiralMods.ToggleEnemies.cfg ].
If you cannot see this file, try running the game with the mod so that the file can be generated by BepInEx, if that doesn't work, verify if you have the mod correctly installed.

## Building

1. Copy files from `./BepInEx/core` to `.libs`
   1. `0Harmony.dll`
   2. `BepInEx.dll`

2. Copy files from `./Lethal Company_Data/Managed` to `.libs`
   1. `Assembly-CSharp.dll`
   2. `UnityEngine.dll`
   3. `Unity.Netcode.Runtime.dll`
   4. `UnityEngine.CoreModule.dll`
   5. `UnityEngine.Physics2DModule.dll`
  
3. Build solution

4. Built mod should be located at `bin/Debug/ToggleEnemies.dll`
